SupportServices Glossary

Key terms and concepts used throughout the platform

📖 Terms

Correlation ID (Operation ID)

A unique identifier that follows a request across all services and components. Used to trace a single user action from frontend → backend → database → service bus → function.

Change Feed (Cosmos)

An ordered, durable stream of INSERT/UPDATE/DELETE events from a Cosmos container. Powers materialized views, search indexing, and event-driven processors.

RU (Request Unit)

Cosmos DB pricing unit. Every read/write consumes RUs. 1 KB read = ~1 RU. Throttling (429) happens when you exceed provisioned RUs.

Partition Key

The field used to distribute data across physical partitions in Cosmos. Must be high-cardinality, evenly-distributed, and query-aligned. Bad partition key = hot partition = throttling.

SLO (Service Level Objective)

Target reliability (e.g., "99.9% availability"). Error budget = 100% - SLO (e.g., 99.9% = 43 minutes downtime/month).

Error Budget

Acceptable downtime within a period. If SLO = 99.9%, error budget = 0.1%. Once spent, stop deploying and focus on reliability.

Circuit Breaker

Resilience pattern: Closed (normal) → Open (failing, reject fast) → Half-Open (test). Prevents cascading failures by fast-failing instead of retrying.

Exponential Backoff

Retry strategy: wait 1s, then 2s, then 4s, then 8s... Prevents thundering herd (all clients retrying simultaneously).

Idempotency

Operation can be executed multiple times with same effect as once. Critical for retries and distributed systems.

Slot Swap (Blue-Green)

Deployment technique: deploy to staging slot, verify, then swap prod ↔ staging traffic. Enables instant rollback if needed.

Health Probe

Endpoint that returns status (healthy/unhealthy). Load balancers use it to remove unhealthy instances from rotation.

KQL (Kusto Query Language)

Query language for Azure Data Explorer and Log Analytics. Used to query traces, metrics, logs, and build dashboards.

OTel (OpenTelemetry)

Vendor-neutral standard for instrumentation (traces, metrics, logs). Language-agnostic SDK for collecting observability data.

Bicep

Modern declarative language for Azure IaC. Compiles to ARM templates. Cleaner syntax than JSON, better readability.

RACI Matrix

Responsibility assignment: Responsible (does work), Accountable (final say), Consulted (asked opinion), Informed (told result).

OKR (Objectives & Key Results)

Goal-setting framework: Objective (what) + Key Results (how to measure). Used for quarterly planning and alignment.

Paved Road

Recommended, tested way to do something. Make right way easy way (templates, tooling, guardrails).

DDD (Domain-Driven Design)

Design approach: model software around business domain. Separate bounded contexts, use ubiquitous language across team.

Service Mesh

Infrastructure layer for inter-service communication. Handles retries, timeouts, circuit breaking, mutual TLS transparently.
